## Как пользоваться

1. Переходим в раздел [**Releases**](../../releases)
2. Качаем `keyboard-smasher.exe`
3. Запускаем `keyboard-smasher.exe`
4. Нажмаем **F9**
5. Если нужно остановить или выйти - **ESC** 

## Статистика

| Параметр | Значение |
|----------|----------|
| Средняя скорость | ~80 000 нажатий/мин |
| Метод отправки | WinAPI `SendInput` (батчевый) |

## Требования

- Windows 10 / 11 (x64)
- Права администратора не требуются
