# Keyboard Smasher

[English](README.md)

## Быстрый старт

1. Скачайте `Keyboard Smasher.exe` со страницы [Releases](../../releases/latest)
2. Запустите файл
3. Выберите нужные группы клавиш
4. Укажите задержки между нажатием и отжатием или включите **No delay** для максимальной скорости
5. Нажмите кнопку **Start**

Для остановки нажмите кнопку **Stop** или клавишу **F1**.

---

## Интерфейс

### Key Groups

Список групп клавиш. Отметьте галочками те группы, которые должны участвовать в нажатиях. Рядом с надписью **Selected** отображается суммарное количество выбранных клавиш.

| Группа | Что включает |
|---|---|
| Mouse | Кнопки мыши |
| Standard Control | Backspace, Enter, Space и др. |
| Navigation | Page Up/Down, Home, End |
| Arrow | Стрелки |
| Editing | Select All, Undo, Insert, Delete |
| Number | Цифры 0–9 |
| Alphabet | Буквы A–Z |
| Numpad | Цифровой блок и операторы |
| Function | F2–F24 (F1 зарезервирован под экстренную остановку) |
| Browser | Кнопки управления браузером |
| Symbols | Символы: `; = , - . / ~` и скобки |
| IME | Клавиши японского/китайского ввода |
| OEM | Специфичные клавиши производителей |
| Gamepad | Кнопки геймпада |
| Other | Прочие системные клавиши |

### Timing

**No delay** — нажатия и отжатия отправляются одним пакетом без задержек. Максимальная скорость, поля задержки блокируются.

**Press / Release** — задержки в миллисекундах между нажатием и отжатием. Диапазон: 0–999 мс. Пустое поле = 0.

### Кнопка старт / стоп

Нажмите кнопку, чтобы запустить. Перед началом идёт обратный отсчёт.

Кнопка меняется на **Stop** (красная) — нажмите её для остановки.

> **Экстренная остановка:** нажмите **F1** в любой момент — скрипт остановится, даже если интерфейс не отвечает.

### Строка состояния

Внизу отображается текущее состояние и таймер:

- `Ready` — готов к запуску
- `Starting in 3...` — обратный отсчёт
- `Running 00h 00m 12s | 128 keys` — работает
- `Stopped after 12s` — остановлен

> При режиме **No delay** или маленьких интервалах строка состояния может подвисать.

## Поведение

- Скрипт **автоматически останавливается**, если окно приложения теряет фокус.
- Настройки (выбранные группы и тайминги) **сохраняются автоматически** и восстанавливаются при следующем запуске.
- Конфиг хранится в `%APPDATA%\keyboard-smasher\config.json`.
